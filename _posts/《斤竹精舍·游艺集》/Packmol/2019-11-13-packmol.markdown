### tethered brush
在71.97x49.86x100的盒子里填充20个P1.pdb分子，并使8号原子在z=0的表面。  
其他说明：文件格式为pdb，输出文件名为P1_20.pdb

```

tolerance 2.0

filetype pdb
output P1_20.pdb

structure P1.pdb
  number 20
  inside box 0. 0. 0. 71.97 49.86 100
  atoms 8
    inside box 0. 0. 0. 71.97 49.86 0
  end atoms
end structure
```
